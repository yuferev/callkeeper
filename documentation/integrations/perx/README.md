# Интеграция c платформой PERX

Вы наверняка знаете что формы можно обрабатывать в обратный звонок. Совместно с коллегами из PERX разработана обработка форм на дилерских сайтах Renault по API.

Теперь вся информация с форм попадает в личный кабинет, а также эту информацию можно проговаривать при соединении оператору для предметного диалога с клиентом.

## Что для этого нужно?

Написать своему менеджеру в CallKeeper с вопросом о подключении. Мы пришлём вам список форм, которые можно подключить к обработке и другую информацию необходимую для подключения. Обязательно наличие API-ключа (apiak) и значение параметра whash.

После этого информацию необходимо будет отправить по адресу технической поддержки сайта или через системы заявок Renault.
